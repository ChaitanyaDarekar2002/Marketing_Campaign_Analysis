# 📊 Marketing Campaign Analysis | Python • SQL Server • Power BI

## 📌 Project Overview

This project analyzes a marketing campaign dataset to uncover customer behavior, purchasing patterns, and campaign performance. The analysis follows a complete data analytics workflow using **Python** for data cleaning and exploratory analysis, **SQL Server** for querying business insights, and **Power BI** for creating an interactive dashboard.

## 🎯 Project Objectives

* Clean and prepare raw marketing data.
* Analyze customer demographics and purchasing behavior.
* Measure campaign effectiveness.
* Identify high-value customer segments.
* Build an interactive Power BI dashboard for business decision-making.

* Python (Pandas, NumPy, Matplotlib)

* Power BI
* Microsoft Excel

## 📂 Project Workflow

### 1. Data Collection

* Imported the Marketing Campaign dataset from Excel.

### 2. Data Cleaning (Python)

* Checked missing values.
* Removed duplicate records.
* Corrected data types.
* Formatted date columns.
* Cleaned inconsistent values.
* Created new calculated columns for analysis.

### 3. Exploratory Data Analysis (Python)

Performed visual analysis including:

* Customer age distribution
* Income distribution
* Total spending by product category
* Purchase channel analysis
* Campaign acceptance analysis
* Marital status distribution
* Education level analysis
* Customer enrollment trend
* Web vs Store purchases
* Correlation heatmap

---

### 4. Business Analysis (SQL Server)

Business questions answered using SQL:

* Total customers
* Average customer income
* Total spending by product category
* Customers accepting each campaign
* Top spending customers
* Purchase channel comparison
* Customers by education level
* Customers by marital status
* Monthly customer enrollment
* Average web purchases
* Average catalog purchases
* Average store purchases
* Recency analysis
* Customers with complaints
* Customer segmentation based on spending

---

### 5. Dashboard Development (Power BI)

Developed an interactive dashboard featuring:

#### KPI Cards

* Total Customers
* Total Revenue
* Average Income
* Average Spending

#### Charts

* Monthly Customer Enrollment (Line Chart)
* Spending by Product Category (Column Chart)
* Purchase Channel Comparison (Bar Chart)
* Campaign Acceptance (Donut Chart)
* Customer Education (Pie Chart)
* Marital Status Distribution
* Income Analysis
* Country/Region Analysis (if available)
* Customer Details Table

#### Interactive Features

* Date Slicer
* Education Filter
* Marital Status Filter
* Campaign Filter
* Drill-through Pages
* Tooltips
* Dynamic KPIs

## 📈 Key Insights
